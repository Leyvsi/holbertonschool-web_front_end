# HTML Advanced - Techium Project

## Overview
This project is a comprehensive deep dive into **HTML5 structure**. The goal is to build a professional, semantic skeleton for a company named **Techium**. 

> **Note:** This project focuses exclusively on HTML. No CSS or styling is applied. The visual result will be unstyled ("ugly"), which is intentional to prioritize structural integrity.

---

## Learning Objectives
Upon completion of this project, I will be able to explain:
* **HTML Guidelines**: Following best practices for clean and valid code.
* **Document Skeleton**: Correct usage of `<!DOCTYPE html>`, `<html>`, `<head>`, and `<body>`.
* **Semantic HTML**: Using tags like `<header>`, `<nav>`, `<main>`, `<section>`, `<article>`, `<aside>`, and `<footer>` to provide meaning to web content.
* **Block vs Inline**: Understanding when to use `<div>` (block) versus `<span>` (inline).
* **Headings**: The importance of SEO and accessibility through hierarchical order (`<h1>` to `<h6>`).
* **Lists**: Creating ordered (`<ol>`) and unordered (`<ul>`) lists.
* **Media Management**: Integrating and choosing between SVG, GIF, PNG, and JPG.
* **Multimedia**: Embedding `<video>`, `<audio>`, and external content via `<iframe>`.
* **Data Organization**: Structuring complex data using `<table>`.
